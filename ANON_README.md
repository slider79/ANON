## Use Cases & User Flows

### Primary Actors

1.  **New User** - First-time visitor wanting to join the network
2.  **Observer** - Read-only user browsing rumors
3.  **Contributor** - User posting new rumors
4.  **Verifier** - User voting on existing rumors
5.  **System** - Automated background processes (consensus, decay, etc.)
